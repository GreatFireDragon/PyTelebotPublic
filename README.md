# PyTeleBot
Учебный проект разработки чат-бота для Телеграмма

Умеет:

- **Присылать:**
  1) кота
  2) человека
  3) анекдот
  5) новости
  6) курс валют
  
- **Редактировать фото**
  1) Блюр
  2) Перевод в ЧБ
  3) Выделение краёв
  4) Отразить
  5) Перевернуть
  6) Превратить в рисунок

- **Игры**
  1) 21
  2) 21 multiplayer
  3) Камень Ножницы Бумага
  4) КНБ multiplayer
  5) Wordle
